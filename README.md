## DarkMode-ANE

Enable Dark Mode on MacOS Mojave for your AIR app.

##### Dependencies
From the command line cd into /example and run:
- OSX
```shell
bash get_dependencies.sh
```

### Prerequisites

You will need:

- macOS 10.14 (Mojave) or greater
- IntelliJ IDEA / Flash Builder
- AIR 19 or greater
- wget on OSX
